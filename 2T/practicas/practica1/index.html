Responsive Web Design con Mobile First

Este diseño adapta su estructura según el tamaña de la pantalla

  • Móvil : Layout vertical, todo en columna
  • Tablet : Grid 2 columnas, menú lateral + contenido/aside apilados
  • Desktop : Grid 3 columnas, todas visibles simultáneamente

Los estilos se cargan en cascada: primero móvil (base), luego tablet, finalmente desktop
